# nodejs-lesson-7
Lesson 7

Для работы приложения перед его зауском необходимо установить базу данных mongoDB и выполнить 2 команды

    > npm install
    > bower install

и после можно запустить одной из следующих команд:

    > node app.js
    > npm start

И открыть браузер по адресу <http://localhost:8000/>.
